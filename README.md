# 在互联网大厂的工作和生活是怎样的？

继我离开事业单位，我在互联网大厂也工作了有近一年半的时间了。这期间的感受“情绪价值”拉满，但是一直未能与大家“对齐”。所以，这篇文章就用来证明我来过、爱过、也痛过，也算是完成了“闭环”。那么仅以本文献给那些处在迷茫中的人儿，无论是想要去体制还是互联网，都可以作为参考。着实也算是“赋能”了一小把，不知是否“符合您的预期”呢？
